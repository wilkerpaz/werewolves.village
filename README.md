# werewolves.village
Jogo de dedução social multiplayer para amigos
Vila dos Lobos é um jogo de dedução social clássico para 4-13 jogadores.

Perfeito para noites em família ou com amigos! O jogo oferece:

🐺 Papéis únicos: Lobos, Vidente, Médico, Bruxa, Sacerdote, Caçador e mais
🎮 Três modos de conectividade:
   • Celular Único: passa o telefone entre jogadores
   • WiFi Local: conecte múltiplos telefones na mesma rede
   • Online: jogue com amigos em qualquer lugar

🌙 Fases dinâmicas: noites de ação silenciosa e dias de votação intensa
🎯 Estratégia pura: sem sorte, apenas lógica, blefe e dedução
⏰ Partidas rápidas: 15-30 minutos por jogo
🔊 Áudio ambiente imersivo

Características:
✓ Sem anúncios
✓ Sem compras dentro do app
✓ Totalmente offline (modo WiFi/Local)
✓ Suporte a português
✓ Interface intuitiva
✓ Histórico de partidas

Ideal para festas, encontros e qualquer ocasião social!
