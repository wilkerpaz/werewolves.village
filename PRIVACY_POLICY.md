# Política de Privacidade - Vila dos Lobos

## Dados Coletados
- Nome de usuário (opcional, armazenado localmente)
- Histórico de partidas (armazenado localmente)
- Logs de conexão (para diagnóstico)

## Armazenamento
- Todos os dados são armazenados **localmente no dispositivo**
- Nenhum dado é enviado para servidor externo (exceto modo online)
- Modo online: nome do jogador e ações são transmitidas via WebSocket

## Terceiros
- Firebase/Google Analytics: Não usamos
- Anúncios: Não há anúncios
- Rastreamento: Não há rastreamento

## Direitos do Usuário
Você pode deletar todos os dados localmente desinstalando o app.
